# SurvBregDiv: Transfer Learning for Cox Models via Bregman Divergence

Transfer Learning for Time-to-Event Modelling via Bregman Divergence.
The package supports both low-dimensional and high-dimensional settings
to leverage external summary statistics or auxiliary data.
